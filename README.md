# Wedding Planner

Complete end-to-end wedding planning app.

Built with Next.js and Tailwind CSS. Current features:

- Home page with countdown to the wedding date
- Guest list with RSVP tracking (saved in localStorage)
- Budget tracker with estimated vs actual costs (saved in localStorage)

## Getting Started

Install dependencies (if needed), then run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
